# ML_Image
Основные возможности проекта:
Классификация изображений - определение типа изображения (пейзаж, портрет, город, абстрактное)
Детекция объектов - поиск лиц и объектов на изображении
Сегментация - разделение изображения на цветовые сегменты
Анализ признаков - извлечение и анализ характеристик изображения
Глубокий анализ - PCA, кластеризация, оценка качества

Технологический стек:
GUI: PyQt5
Компьютерное зрение: OpenCV, Pillow
Машинное обучение: scikit-learn (SVM, Random Forest, KNN, Neural Networks)
Обработка данных: NumPy, scikit-image

ML алгоритмы:
Классификация: SVM, Random Forest, K-Nearest Neighbors, Neural Network (MLP)
Кластеризация: K-means
Анализ: PCA для уменьшения размерности
Извлечение признаков: цвет, текстура, яркость, контраст, энтропия
